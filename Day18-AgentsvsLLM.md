## â“ What Is an LLM?
A Large Language Model (LLM) is a model trained to generate text based on patterns learned from large datasets.

LLMs take an input prompt and produce an output response, but they do not take actions, make decisions, or interact with external systems on their own.

LLMs are reactive systems.

## â“ What Is an Agent?
An agent is a system that uses an LLM as its reasoning component but adds the ability to:
- Decide what to do next
- Use tools or APIs
- Maintain state or memory
- Act toward a goal
